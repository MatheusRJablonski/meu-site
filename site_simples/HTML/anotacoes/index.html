Elementos

como usar o <html>

responsavel por todo o conteúdo da página e às vezes é conhecido como o elemento raiz.

o que é <head>

head serve como um recipiente de tudo o que você deseja incluir em uma página HTML que não é o conteúdo que você quer mostrar para quem vê sua página.

o que é <body>

Contém todo o conteúdo que você quer mostrar ao público que visita sua página, seja texto, imagens, vídeos, jogos, faixas de áudio reproduzíveis ou qualquer outra coisa.

o que é <header>

pode conter os elementos de um cabeçalho mas tambem pode conter elementos como logo,seções,etc.

o que é <section>

representa uma seção genérica contida em um documento HTML, geralmente com um título, quando não existir um elemento semântico mais específico para representá-lo

o que é <div>

é um container genérico para conteúdo de fluxo, que de certa forma não representa nada. Ele pode ser utilizado para agrupar elementos para fins de estilos (usando class ou id), ou porque eles compartilham valores de atributos, como lang.

o que é <footer>

assim como o cabeçalho temos o rodapé no html, sua função é criar um rodapé.

o que é <h>

serve para indicar um titulo ou subtitulo, os seguintes tamanhos são:

tilulo h1

titulo h2

titulo h3

titulo h4

titulo h5
titulo h6
titulo h7

o que é <hr>

serve para criar linhas horizontais que podem ser usadas para dividir o conteudo.
assim como esta sendo feito com estas linhas vermelhas.
para mudar a cor da linha pode-se usar o comando <hr color="red">
